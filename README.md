# naver_reservation
- 본 프로젝트는 네이버 부스트코스의 정책으로 해당 공개된 repository에 코드를 올릴 수 없습니다.

# 제작 목적

- front end, back end의 전반전인 개발 능력 향상
- 웹에 대한 이해
- Spring Framework 개발 경험
- WEB API 개발 경험
- DATA BASE 전처리 능력 향상
- 실무에 도움 되는 클린 코드 작성 학습
- 실무자들의 코드 리뷰를 통한 개발 능력 향상

# 주요 기능

1.메인 화면
2.예약 상세
3.상품 예약
4.예약 상품 조회
5.상품 리뷰

# 결과

- 모든 프로젝트 섹션을 Pass 받고 수료증을 발급 받았다.
- 기본적인 웹 개발을 할 수 있는 실력을 키울 수 있었다.
- 블로그를 통해서 오류, 이슈, 정보들을 공유하는 시발점이 되었다. (blog : https://hwi-programing.tistory.com/)
